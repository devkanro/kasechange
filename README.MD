Multiplatform Kotlin library to convert strings between various case formats

# Supported Case Formats
- SCREAMING_SNAKE_CASE
- snake_case
- PascalCase
- camelCase
- TRAIN-CASE
- kebab-case
- UPPER SPACE CASE
- Title Case
- lower space case
- UPPER.DOT.CASE
- dot.case
